# Credits

Thinkers Roundtable was inspired by Huashu's Nuwa Skill:

https://github.com/alchaincyf/nuwa-skill

Nuwa helped popularize the idea of turning a person's thinking patterns into reusable agent instructions: mental models, decision heuristics, and expression patterns.

This repository is a separate free project focused on one use case: structured decision pressure-testing through a roundtable of decision lenses.

No endorsement by the people named in this project is implied.
